# fcards: cli flashcards

## installation

``` bash
git clone https://github.com/bwally0/fcards
cd ./fcards
pip install .
```

## usage

``` bash
fcards ./examples/cse470-final.txt
```

## .txt format

to create new flashcard set, create a text file with the following format:
```
Q>your question here
A>your answer here

Q>your question here
A>your answer here

and so on ...
```
